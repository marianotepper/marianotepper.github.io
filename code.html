Dask library

Dask enables parallel computing through task scheduling and blocked algorithms. This allows developers to write complex parallel algorithms and execute them in parallel either on a modern multi-core machine or on a distributed cluster. See the full dask documentation here.

I am a developer of the linear algebra module in Dask.

Kinect camera

Camera-projector calibration to perform a "real" augmented reality. Part of the Cámara Lúcida project (demo videos available at the project website). The functionality has been added to the RGBDemo software.
